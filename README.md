## Steps

1. **Virtual Environment Setup:**

     ```bash
     python -m venv Effectively-Using-AI-Agents-for-Customer-Query-Management
     ```

     - Windows:
       ```bash
       Effectively-Using-AI-Agents-for-Customer-Query-Management\Scripts\activate
       ```
